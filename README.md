# ARCADE-HUB 
Fluxo:
🔤 Tela de nome — digita seu nome de piloto
🎮 Menu — escolhe um dos 3 jogos
🏆 Game Over — retry ou volta pro menu
3 jogos incluídos:
🚀 Asteroid Storm — nave + asteroides + escudo
🐍 Neon Snake — cobra clássica com grid neon
🧱 Breakout X — rebater a bola nos blocos coloridos
Features extras:
Nome personalizado aparece no menu e no recorde
Recorde salvo por jogo na sessão
Controles mobile adaptados pra cada jogo
Botão [ ← MENU ] pra voltar a qualquer hora
Sons únicos em cada jogo
Tudo compartilha o fundo estrelado animado
